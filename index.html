Event Map

General description.

Users can import a map of a place (such as uw's) and mark the important addresses on it, such as kane hall, such as ct, such as dis market, and then freely edit the content inside, adding business hours, comprehensive evaluation, photos, basics Information, if it is a scenic spot, you can write history or something. To put it bluntly, it is a web tool that can make and interact with maps

targeted users

Our app will create an event map for students who struggles to arrange their events and estimate the time they run from one event to another.

Designated functions

  • import a map of any place
  • freely add a description of the place
  • upload photos of it to change the "background" of one spot on the map
  • set up an alarm clock to remind them of their business hours
  • update what they wrote and keep multiple version of a place's description

Designated Functions

  • Import multiple maps or create one. Maps can be public and edited by anyone
  • Mark, edit, and add descriptions of places on maps
  • Freely add public or private events and mark them on maps, create personalized event cards
  • Create differnt forms of schedule for one's events
  • Set up alarm clocks to remind users of their events
  • Check on other users' schedule and events.
  • Search through or filter events
  • Share photos and other details using the event cards
